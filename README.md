# üè™ Full-Stack Inventory Management System

This is a complete, from-scratch inventory management application. This system provides a clean, responsive interface to manage products, track stock levels, and view key inventory statistics.

It's built with a React frontend (styled with standard CSS), a Node.js/Express backend API, and a MySQL database.


## ‚úÖ Features

* **Real-time Dashboard:** View key stats at a glance, including total products, total stock value, low stock items, and out-of-stock items.
* **Full Product CRUD:** A complete Create, Read, Update, and Delete workflow for all your products.
* **Dynamic Stock Status:** Products are automatically tagged as "In Stock," "Low Stock," or "Out of Stock" based on quantity.
* **Live Search:** Instantly filter products by name or SKU.
* **RESTful API:** A well-structured backend API built with Express to handle all data operations.
* **Database Automation:** The server automatically creates the `products` table and inserts sample data on its first run.
